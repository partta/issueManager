###Install
Step 1 Clone this repo somewhere in your machine
Step 2 install node dependencies ``` npm install ```
Step 3 ``` cd client ``` and run ``` npm install ```
Step 4 ``` cd .. ``` to return to root folder
Step 5 run ``` npm start ```
